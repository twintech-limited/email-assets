Konnect xD
v1.9.3

Release Notes

A major overhaul of the 3D Viewer experience — redesigned project selector, improved mobile support, upgraded rendering engine, large model upload support, and comprehensive quality improvements.

March 16, 2026 For QMW (McDermott Wuchuan)
Features Quality Improvements

New Features & Enhancements

⬆️

Large Model Upload Support

Performance

The backend upload pipeline has been enhanced to support uploading significantly larger 3D models. Previously, large model uploads could fail due to property download timeouts. The upload and model reconstruction process has been reworked to reliably handle large, complex models — ensuring your full-scale project models upload and process successfully.

🔀

Redesigned Project & Model Selector

New

The project and model selection experience has been completely redesigned. A single button now opens a dedicated panel where you can browse available models, switch between projects, and quickly navigate back to the Model Selection Page — all without leaving the 3D Viewer. The new selector also works seamlessly on mobile devices including iPhone and Samsung Galaxy Fold.

⚡

Faster Model Loading with Caching

Performance

Projects and models are now pre-loaded in the 3D Viewer dropdowns. Thanks to frontend caching, opening the model selector a second time is significantly faster — no more waiting for the list to reload each time you switch views.

🔧

Upgraded 3D Rendering Engine

Performance

The underlying Autodesk Forge Viewer has been updated to version 7.115.2 (latest), bringing improved rendering performance, better stability, and enhanced compatibility with complex 3D models.

🎨

Improved 3D Viewer Layout

Improved

The panel-less version of the 3D Model Viewer has received design improvements, ensuring a cleaner and more consistent visual layout across desktop and mobile views.

📐

Compact Panel Menu

Improved

The side panel ribbon now uses shorter, more space-efficient labels — Take-Off, Lifecycle, POI, and Packages — freeing up valuable screen real estate for the 3D model. Full names are still displayed in the panel headers when expanded.

📱

Mobile View Enhancements

Mobile

The mobile experience has been comprehensively improved based on user feedback:

  • Smart panel toggle — On smaller screens (below 1024px), the tools panel is now hidden by default to maximize the 3D view. A toggle button with smooth animation allows you to expand it when needed.
  • Responsive layout fixes — Navigation, model selector, and tool panels have been refined for better usability on phones and tablets.
🗺️

Guided Tour Improvements

Improved

The in-app guided tour has been refined based on detailed user feedback:

  • Improved alignment and positioning of tour highlights
  • Corrected step descriptions for context menu actions (Isolate, Hide, Focus, Pivot, Color, Find Relationships, etc.)
  • Fixed highlighting to only target relevant UI elements during each tour step
🖱️

Direct Right-Click in Search Results

New

You can now right-click directly on items in the search results to access context menu actions — Isolate, Hide, Focus, Pivot, Color, Find Relationships, and more — without having to first locate the item in the Explorer tree.

✨

Top Bar & UI Polish

Improved

A round of visual polish has been applied to the top navigation bar:

  • User profile menu placement improved
  • Hover states refined — no more unintended highlight effects on the username
  • Project selection screen layout cleaned up
  • Top bar icons now show the full hover area including text labels
🌐

Chinese Language Support Expanded

Localization

Chinese (中文) translation has been expanded to cover the User Profile screen, including profile settings, project/role tables, pagination, and all action menus. Additionally, toolbar tooltips (Orbit, Pan, Zoom, First Person, Measure, Section Analysis, Markups) and measurement sub-menus are now fully translated.

🔒

Improved Access Control Feedback

New

When a user tries to access a project they don't have permission to view, the system now displays a clear "Forbidden" page with an appropriate message — instead of showing a blank screen.

🏷️

Tag Import — Duplicate Node Prevention

Data Quality

The tag import pipeline now automatically normalizes all tag and hierarchy names by trimming leading/trailing spaces and replacing non-standard whitespace characters. This prevents duplicate nodes from appearing in the Explorer (e.g., "Instrumentation Equipment" and "Instrumentation Equipment " are now correctly merged into a single node). Affected fields include Discipline, Area/Category, Tag Type, and Tag Name.

📦

QMW Tag Generation

Data

Tag generation has been completed for the Q0248-Ruya and Q0248-LA projects in preparation for this release.

Quality & Reliability Improvements

Attribute Configuration

Duplicate PIMS attributes resolved

The attribute configuration page previously showed duplicate PIMS attributes for certain projects. These duplicates have been eliminated.

3D Viewer

Cloud streaming models now load correctly

An issue that prevented cloud-streamed models from opening has been resolved.

3D Viewer

Global Search restored

The search function in the 3D Viewer was returning no results even for valid queries. Search now works as expected.

3D Viewer

Piping visibility in transparency mode

When coloring isometrics and applying transparency to Structural elements, piping objects no longer disappear. (Note: some transparency limitations remain due to the underlying Forge rendering engine.)

User Management

Edit User page fixed

System administrators editing a user's profile no longer encounter a blank page.

User Management

Email case sensitivity resolved

User email addresses are now treated as case-insensitive, preventing duplicate accounts from being created (e.g., "user@example.com" and "User@example.com" are now recognized as the same user).

User Profile

Language change confirmation

Changing your preferred language in the profile settings now displays a confirmation message after saving, instead of silently applying the change.

Page Titles

Dynamic page titles corrected

Browser tab titles now correctly reflect the current page across Ingestion, Upload PDF Documents, Data To Node, and Project views.

Model Upload

Model uploads restored

An issue that prevented 3D model uploads has been resolved. Models can now be uploaded successfully.

Take-Off Manager

Panel no longer closes after deleting a preset

Previously, deleting a Take-Off Manager preset would automatically close the entire panel. The panel now stays open so you can continue working.

Take-Off Manager

Preset creation when many presets exist

Fixed an issue where creating a new Take-Off Manager preset would fail if a large number of presets already existed in the project.

Spool Tracker

PBS node creation on upload

Uploading a Spool Tracker file now correctly creates the associated PBS (Product Breakdown Structure) nodes.

Release Summary

Version 1.9.3 delivers a comprehensive set of new features and quality improvements across the Konnect xD platform, with a particular focus on the 3D Viewer experience, mobile responsiveness, and Chinese language support.
